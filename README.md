# X OpenClaw Threads 技能

在 X（推特）上搜索与 OpenClaw、AI 相关的热门推文，并整理成 **Threads** 风格帖子的技能包。

## 功能概要

- 按关键词在 X 上检索最新/热门内容
- 过滤与归档到按日期的素材 Markdown
- 输出符合规范的 Threads 改编稿（简体中文）

## 使用说明

详细步骤、日期规则、图片/视频处理与输出禁忌见同目录下的 **`SKILL.md`**。
